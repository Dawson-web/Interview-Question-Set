- 缓存穿透、缓存击穿、缓存雪崩

- 高并发情况处理
- 前端实例和后端实例数量，怎么做转发
- Nginx 负载均衡实现
- 计网状态码
- TCP 三次握手四次挥手，为什么三次握手

### 面试内容

> 两次腾讯前端面试（已入职），先面 TEG 后面 CSIG。

**腾讯 TEG 一面**（30min）
- 自我介绍
- 简历中哪个项目投入最深？做了多长时间
- CSS 中 position 属性有哪些有效值
- CSS 中 display 属性有哪些有效值
- 盒模型有几种，分别是什么
- 数据库的主键约束和外键约束
- 什么是事务？事务的原子性是什么
- MySQL 中 select 的具体执行过程
- 算法题（口述思路）：乱序不重复 1-100 数组去掉一个数字，如何找出，要求空间复杂度最小
- React 框架了解吗
- React 和 Vue 的区别
- 两个框架的数据绑定方式
- React 是 MVVM 还是 MVC 框架
- 什么是 MVVM 框架
- 职业规划
- 反问

**腾讯 CSIG 一面**（60min）
- 自我介绍
- 小程序日活多少
- 小程序开发用什么框架
- 小程序的生命周期函数
- null 和 undefined 的区别
- 对 Node.js 了解多少
- ES6 相比 ES5 有哪些新特性
- 手撕：哈希表
- 用过哪些数据库？InfluxDB 有什么特点
- Linux 操作系统指令
- HTTP 和 HTTPS 区别
- TCP 三次握手
- 快速排序算法原理
- 反问

**腾讯 CSIG 二面**（40min，围绕项目）
- 哪个项目投入最深，介绍项目，负责哪些部分
- 用户提交表单到服务器再到渲染在页面上的整个过程，用到了哪些协议
- 了解热加载吗
- 页面加载慢有哪些原因
- 性能优化方案
- 为什么有时候 WebP 格式的图片显示不出来
- Edge 浏览器和 Chrome 浏览器区别
- 图片有哪些压缩方法，压缩算法原理和对比
- 雪碧图中不同尺寸的图片如何定位
- 图片懒加载原理
- 如何监控服务器的剩余资源
- 如何减少数据库的读写次数
- 多个设备上的本地缓存如何同步
- 智力题：100 本书，每次只能拿 1-5 本，你先拿，如何保证拿到最后一本
- 反问
