### 面试内容

- 项目主要解决的业务问题是什么？上线后谁在用
- 审核和上下架模块在权限控制上有没有联动逻辑
- 字段不能修改是怎么判断的？状态变化后哪些字段可以改
- 审核的状态设计
- ProTable 中分页、筛选、操作区是自己写的吗？有没有封装复用组件？分页功能如何实现
- 有看过 ProTable 底层吗？弹窗组件、Form 组件
- 封装的弹窗表单，从表格点击编辑时字段异步加载怎么办？Modal 打开后请求还没回来
- AntD 的 Form 是怎么绑定字段的？field 如何注册到父表单？字段更新触发机制
- ProTable 底层 request 逻辑在哪一层触发？分页和筛选怎么 merge 到最终请求
- 项目上线后用户量多少？有做监控告警吗，怎么监控的
- XML 渲染系统如何支持动态组件渲染？多层嵌套怎么处理
- 合并 XML 文件怎么保证结构不冲突
- 什么是跨域，怎么处理跨域，跨域返回的网络码是多少，4xx 和 3xx、5xx 的区别
- 如果要把后台项目变成 SaaS 化平台，权限、主题样式、数据隔离前端会考虑哪些点
- useEffect 和 useLayoutEffect 的执行时机差别
- React 的状态更新是异步的吗？哪些场景下是同步的
- React 事件机制，为什么不是原生的
- JS 的垃圾回收机制
- 手撕：合并两个升序链表
