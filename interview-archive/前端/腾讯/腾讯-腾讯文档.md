#### 📋 项目经验
- 你xx后台那个项目，主要解决的业务问题是什么？上线之后是谁在用？
- xx审核和xx上下架你都做了，那这两个模块在权限控制上有没有什么联动逻辑？
- 你们当时做xx审核的时候设计过哪些状态
- 你说用了ProTable做复杂表格，那里面分页、筛选、操作区这些是自己写的吗？有没有自己封装过一些复用的组件？分页功能是如何实现的
- 你封装的弹窗表单，如果是从表格里点击某条数据进入编辑，字段异步加载怎么办？Modal打开之后请求还没回来这种
- 这个最后上线完大概有多少个用户量？有做监控告警么，怎么监控的？
- 如果你现在要把这个后台项目变成SaaS化平台，多家企业共用，权限、主题样式、数据隔离这些你前端会考虑哪些点

#### ⚛️ React
- useEffect和useLayoutEffect 的执行时机具体有什么差别
- React 的状态更新是异步的吗？在哪些场景下是同步的
- React事件机制，它的事件为什么不是原生的

#### 🌐 网络与安全
- 什么是跨域，怎么处理跨域，跨域返回的网络码是多少，四叉叉和三叉叉有什么区别吗？还有五叉叉

#### 💻 JavaScript 基础
- 说一下JS的垃圾回收机制

#### 🗃️ 数据结构与算法
- 手撕：将给定的两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接这两个链表的所有节点组成的

#### 🔧 组件库/框架原理
- 有看过它底层么，比如说这个弹窗组件，form组件
- AntD的Form是怎么绑定字段的？你知道field实际上是怎么注册到父表单上的吗？字段更新触发机制你了解吗
- ProTable底层request逻辑是在哪一层触发的？分页和筛选是怎么merge到最终请求里的

#### 📝 其他
- 有字段不能修改，是怎么判断的？状态变化后哪些字段可以改？哪些不能？
- XML渲染系统，是怎么支持动态组件渲染的？比如有这种表单或者多层嵌套怎么处理的？
- 当时合并XML文件怎么保证结构不冲突
